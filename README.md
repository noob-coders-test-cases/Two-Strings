# Two-Strings
Hackerrank Solution
Problem Statement:
Given two strings, determine if they share a common
   substring.
 A substring may be as small as one character.

For example, the words "a", "and", "art" share 
   the common substring . 
The words "be" and "cat" do not share a substring.

Sample Input : 
2
Hello
Hi
Sample Output:
YES

Sample Input:
2
Hello
Waste
Sample Output:
NO
